INSTALLATION OPTIONS
--------------------

Option 1 - Automated (Recommended):
1. Extract ZIP file
2. Right-click Install.bat
3. Select "Run as administrator"
4. Follow on-screen instructions

Option 2 - Manual:
1. Extract ZIP file
2. Copy RssVisualizerScreensaver.exe to C:\Windows\System32\
3. Rename to RssVisualizerScreensaver.scr (optional)
4. Configure via Windows screensaver settings

QUICK START
-----------

1. Install using one of the methods above
2. Right-click Desktop ‚Üí Personalize ‚Üí Lock screen ‚Üí Screen saver
3. Select "RssVisualizerScreensaver" from dropdown
4. Click "Settings" to configure (optional - comes with good defaults)
5. Click "Preview" to test
6. Set wait time and click "OK"

TROUBLESHOOTING
---------------

If screensaver doesn't start:
‚Üí Check file extension is .scr
‚Üí View C:\Windows\Temp\rss_viz.log
‚Üí Check Windows Event Viewer

If feeds don't load:
‚Üí Verify internet connection
‚Üí Check feed URLs in browser
‚Üí Review log file for specific errors
‚Üí Try default feeds via "Restore Defaults"

If word cloud shows unwanted terms:
‚Üí Settings ‚Üí Word Cloud
‚Üí Add terms to exclude (comma-separated)
‚Üí Save and restart screensaver

UPGRADING
---------

From Beta/Preview:
1. Close screensaver if running
2. Delete old config: %APPDATA%\RssVisualizerScreensaver\
3. Install v1.0.0 using instructions above
4. Reconfigure settings in Settings dialog

TECHNICAL DETAILS
-----------------

Built with:
- .NET 8.0 (bundled, self-contained)
- WebView2 for rendering
- WPF for Windows UI
- System.ServiceModel.Syndication for RSS parsing

Deployment:
- Single-file executable
- Self-contained (no .NET runtime required)
- ReadyToRun compilation for faster startup
- Native libraries embedded

Architecture:
- C# backend for feed management
- HTML/CSS/JavaScript for visualization
- Token-based templating for dynamic content
- In-memory caching with refresh timers
